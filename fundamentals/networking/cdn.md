# CDN (Content Delivery Network)

Network of servers distributed geographically to serve content faster.

---

## Why Use It

- Reduces latency — content served from nearest server
- Offloads static content from origin server

---

## What Gets Cached

- Images, videos
- CSS, JS files
- Static HTML

---

## Trade-offs

| Factor | Details |
|---|---|
| Load Time | Faster |
| Origin Server Load | Reduced |
| Cache Invalidation | Complex |
| Cost | Increases |

---

## Real-World Usage

Netflix, Instagram — serve static and media content through CDNs globally.

---

**CDN = bring content closer to users. Trade-off is cache management and cost.**
