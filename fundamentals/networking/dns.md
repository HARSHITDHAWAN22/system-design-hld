# DNS (Domain Name System)

Translates domain names into IP addresses.

---

## How It Works
User enters domain → DNS resolves to IP → Request hits server
---

## Why It Matters

- Makes internet human-readable
- Enables service discovery in distributed systems

---

## Trade-offs

| Factor | Details |
|---|---|
| Latency | DNS lookup adds time |
| Caching | Speeds it up but adds staleness risk |

---

**DNS = phonebook of the internet. Every request starts here.**
